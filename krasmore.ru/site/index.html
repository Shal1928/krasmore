Услуги

Above, you find an example of a beautiful advanced background image slideshow, recreated after the effect as seen on the website from Philadelphia. It features changing text and playback controls. When the animation doesn't seem smooth, the images might not be loaded. This script can perfectly be combined with an image preloading plugin to get rid of this effect.

This technique heavily relies on jQuery combined with CSS. Transparent PNG's are used to create the see-through effect in the menu and text background.

Don't forget to read the article to see how it works and download the source code. Tested and works on Firefox, Safari, Chrome and Opera.
